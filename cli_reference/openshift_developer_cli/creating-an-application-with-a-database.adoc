= Creating an application with a database

//include::modules/common-attributes.adoc[]
:context: creating-an-application-with-a-database
toc::[]

This module describes how to deploy and connect a database to a frontend application.

.Prerequisites

* {odo-title} is installed.
* `oc` client is installed.
* You have a running OpenShift cluster. Developers can use link:https://cloud.redhat.com/openshift/install/crc/installer-provisioned?intcmp=7013a000002CtetAAC[CodeReady Containers (CRC)] to deploy a local {product-title} cluster quickly.
* Service Catalog is enabled.
* Download an example application using `$ git clone https://github.com/openshift/nodejs-ex` and create a project with it.
