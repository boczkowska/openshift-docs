{odo-title} allows creating a multiple-component application, modifying it, and linking its components in an easy and automated way.

This module describes how to deploy a multiple-component application - a shooter game. The application consists of a frontend Node.js component and a backend Java component.

.Prerequisites

* {odo-title} is installed.
* You have a running OpenShift cluster. Developers can use link:https://cloud.redhat.com/openshift/install/crc/installer-provisioned?intcmp=7013a000002CtetAAC[CodeReady Containers (CRC)] to deploy a local {product-title} cluster quickly.
* Maven is installed
* Download example applications using:
** `$ git clone https://github.com/openshift-evangelists/Wild-West-Backend backend`
** `$ git clone https://github.com/openshift-evangelists/Wild-West-Frontend frontend`
+
and create a project with it.
